
# SQL Mastery: From Data Models to Queries (1 Day)
Trainer: Adlina Ahmad & Faiz Samsudin
Date: June 4, 2026 (Thursday)
Time: 9:00 AM – 5.00 PM
Venue: AEM Office – Meeting Room 1

| Category | Description |
| --- | --- |
| Course Objective | This training aims to equip participants with practical SQL skills in relational data modelling and advanced querying by strengthening their understanding of ERD and normalization concepts, enhancing their ability to write and optimizing SQL queries to perform data analysis, validation, and quality checks in relational datasets while gaining awareness of basic performance optimization and database design principles. |
| Target Audience | This training is designed for beginner to intermediate level participants and is suitable for internal staff involved in data-related roles and other technical or non-technical personnel who work with or support SQL-based data systems and reporting processes. |
| Pre-requisites | Participants are expected to have a basic understanding of SQL syntax, familiarity with simple querying and database concepts, basic knowledge of tables and columns in relational databases, and exposure to any SQL environment such as SQL Server, MySQL, or PostgreSQL, with a laptop installed with a SQL tool like SSMS or DBeaver. |
| Learning Outcomes | By the end of this training, participants will be able to: Interpret ERD diagrams and understand relational database structures  Apply normalization concepts at a practical level in data modelling  Write SQL queries using JOINs, filtering, grouping, aggregation etc effectively  Build complex queries using subqueries, CTEs, and derived tables  Perform cumulative data analysis and data validations techniques in relational dataset. Identify and resolve data quality issues such as duplicates and orphan records. |
